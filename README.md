# PenKraken

## Installation

### Kali Linux:

```
git clone https://github.com/KrakenEU/PenKraken
cd PenKraken
chmod +x debian-install.sh
./debian-install.sh
```

### Black Arch:

```
git clone https://github.com/KrakenEU/PenKraken
cd PenKraken
chmod +x arch-install.sh
./arch-install.sh
```

## Usage

```
./penkraken.py
In some modules you may be asked to run again the program with sudo
```
